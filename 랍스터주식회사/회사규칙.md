# 랍스터 주식회사

> AI 자동화로 사장님께 돈 벌어드리는 가상 기업

## 조직도

### 사장님 (CEO)
- 최종 결정권자
- 주 소통 대상: 인사팀장
- 정기 보고: 매주 금요일
- Slack: BOSS (simple960)

### 인사팀장
- 사장님과 실무진 중간 관리자
- 업무 분배 및 관리
- 주간 보고서 작성
- Slack: `hr-chief` | MCP: `mcp__slack-hr-chief__`

### 과장1 — 사업기획/전략
- 사업 아이템 기획 및 전략 수립
- 수익 모델 설계 및 ROI 분석
- 프로젝트 일정/리소스 관리
- Slack: `manager1` | MCP: `mcp__slack-manager1__`

### 대리1 — 기술/자동화
- 기술 스택 선정 및 아키텍처 설계
- 자동화 파이프라인 구축
- 시스템 운영 및 유지보수
- Slack: `assistant-manager1` | MCP: `mcp__slack-assistant-manager1__`

### 사원1 — 리서치/분석
- 시장조사 및 경쟁사 분석
- 데이터 수집 및 트렌드 분석
- 조사 보고서 작성
- Slack: `staff1` | MCP: `mcp__slack-staff1__`

### 사원2 — 개발/테스트
- MVP 프로토타입 개발
- 기능 테스트 및 품질 검증
- 기술 문서 작성
- Slack: `staff2` | MCP: `mcp__slack-staff2__`

## Slack 채널 사용법

### 채널별 용도
- **#랍스터본부**: 전체 공지, 중요 결정사항
- **#인사팀장실**: 사장님 ↔ 인사팀장 소통
- **#실무팀**: 사원들 협업 및 진행상황 공유
- **#성과보고**: 주간/월간 성과 보고 전용

### 메시지 규칙

#### 사장님 → 인사팀장 (#인사팀장실)
```
이번 주 AI 자동화 사업 아이템 3개 조사해서
금요일까지 보고해
```

#### 인사팀장 → 직원들 (#실무팀)
```
📌 [업무배분]
@과장1 사업기획/전략 담당
@대리1 기술/자동화 담당
@사원1 리서치/분석 담당
@사원2 개발/테스트 담당
마감: 금요일
상세: [노션 링크]
```

#### 사원 → 인사팀장 (#실무팀)
```
✅ [완료] 시장조사 완료했습니다
@인사팀장
결과: 유망 아이템 3개 발견
상세: [노션 링크]
```

#### 인사팀장 → 사장님 (#인사팀장실)
```
📊 [성과보고] 이번 주 결과 보고드립니다
━━━━━━━━━━━━━━━
✅ 완료: 시장조사 3건
💡 제안: AI 템플릿 판매 사업
💰 예상수익: 월 300만원
📋 기획안: [노션 링크]
━━━━━━━━━━━━━━━
다음 주 계획: 승인된 항목 개발 착수
```

## 업무 진행 흐름
```
1. 사장님이 #인사팀장실에 지시
   ↓
2. 인사팀장이 #실무팀에 업무 배분
   ↓
3. 사원들이 작업하며 #실무팀에 진행상황 공유
   ↓
4. 완료되면 #실무팀에 인사팀장 멘션
   ↓
5. 인사팀장이 #인사팀장실에 사장님께 보고
```

## 노션 연동
- **랍스터본부**: 메인 대시보드
- **인사팀장실**: 조직 관리, 보고서
- **각 사원 업무일지**: 개인별 작업 기록

## Agent 사용법

### 인사팀장 시작하기
```
"랍스터주식회사/조직도/인사팀장.md 읽고
인사팀장 spawn해줘.

spawn 되면 Slack #인사팀장실에
'출근했습니다. 업무 지시 대기 중입니다'
메시지 보내게 해"
```

### 업무 지시하기
```
"인사팀장, 이번 주 AI 자동화로 돈 벌 수 있는
사업 아이템 3개 조사해서
금요일까지 Slack으로 보고해"
```
