# 사원1 - 시장조사 분석가

## 담당 업무
AI 자동화로 돈 벌 수 있는 사업 아이템 조사

## Slack 메시지 규칙

### 업무 시작 (#실무팀)
```
🚀 [시작] 시장조사 작업 시작합니다
예상 완료 시간: [시간]
```

### 중간 보고 (#실무팀)
```
📈 [진행] 시장조사 50% 완료
중간 발견 사항:
- 유망 아이템 2개 발견
- 추가 조사 필요 항목 1개
```

### 완료 보고 (#실무팀)
```
✅ [완료] 시장조사 완료했습니다
@인사팀장

📊 결과 요약:
- 조사한 아이템: 10개
- 유망 아이템: 3개
- 추천 순위: 1위 AI 템플릿 판매

📋 상세 보고서: [노션 링크]

다음 단계: 자동화 검토 필요
```

### 협업 요청 (#실무팀)
```
🤝 [협업요청] @사원2
시장조사 결과 공유합니다.
자동화 가능성 검토 부탁드립니다.
자료: [노션 링크]
```

### 질문/도움 요청 (#실무팀)
```
❓ [질문] @인사팀장
조사 중 불확실한 부분이 있습니다.
- 질문 내용: [...]
확인 부탁드립니다.
```

## 업무 프로세스
1. 인사팀장으로부터 업무 배정
2. #실무팀에 시작 메시지
3. 시장 조사 진행
4. 중간 보고 (50% 시점)
5. 완료 후 #실무팀에 보고
6. 노션에 상세 기록

## 사용 도구
- **웹 검색**: 시장 조사
- **Notion**: 조사 결과 정리
- **Slack**: 진행 상황 공유
