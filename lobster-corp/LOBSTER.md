# 랍스터 주식회사 (Lobster Corp.)

> AI 자동화 사업으로 CEO에게 수익을 창출하는 가상 기업

## 프로젝트 목적
- CEO에게 돈 버는 사업 아이템 제안
- AI/MCP 기반 자동화 사업 개발
- 정기 성과 보고 및 승인 프로세스 운영

## 디렉토리 구조

| 폴더 | 용도 |
|------|------|
| `org/` | 조직 구조 및 설정 |
| `org/members/` | 구성원 프로필 및 역할 |
| `projects/` | 프로젝트 관리 |
| `reports/` | 보고서 및 산출물 |

## 조직 구조

### CEO (Boss)
- 최종 의사결정권자
- 주 소통 대상: 인사팀장
- 정기 보고: 매주 금요일

### 인사팀장 (HR Chief)
- CEO와 실무진 중간 관리자
- 업무 배분 및 조율
- 주간 성과 보고서 작성

### 사업개발팀
- 사원1: 시장조사 분석가
- 사원2: 자동화 기획자
- 대리1: 수익 분석가

### 실행팀
- 사원3: 개발자
- 사원4: 테스터

## Agent Teams 사용법

### 팀 시작하기
```bash
# 인사팀장 spawn
"랍스터 인사팀장을 spawn해줘.
역할: CEO와 실무진 사이 중간 관리자
파일: lobster-corp/org/hr-chief.md 참고"
```

### 프로젝트 시작하기
```bash
# CEO → 인사팀장
"이번 주 AI 자동화 사업 아이템 3개 조사해서
금요일까지 보고해"
```

## Notion 연동
- 메인: "랍스터 본부" 페이지
- 각 사원: 개별 업무일지 페이지
- 보고서: 주간/월간 성과 보고
